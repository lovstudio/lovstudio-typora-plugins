# Image Fit Viewer for Typora

This is a small local plugin for `typora-community-plugin`.

## Behavior

- Images in the Typora editor are capped at `240px` high.
- Double-click an image to show it in a full-window overlay.
- Edit the image alt text in the bottom bar and click save.
- Press `Esc`, click the backdrop, or double-click the enlarged image to close it.

## Install

After `typora-community-plugin` is installed, copy this folder to:

```text
~/Library/Application Support/abnerworks.Typora/plugins/plugins/local.image-fit-viewer
```

Then enable `Image Fit Viewer` in Typora's community plugin settings.
Restart Typora after installing or updating the plugin.
